Roadmap 2026 Platform Themes¶
  1. Operate a unified application hosting platform across public cloud and GC Private Cloud environments
  2. Build sustainable platform practices that reduce cognitive load for both platform and application teams
  3. Enable departments through a community-driven, CNCF-aligned platform with clear guardrails and autonomy
The roadmap below highlights the major platform epics and strategic initiatives planned or already in progress for Aurora in 2026. Items are grouped by quarter to provide directional visibility into how work is expected to progress across the year, while allowing the flexibility required to operate a shared, enterprise-scale platform. Each quarter reflects a concentration of effort rather than a fixed commitment to delivery dates.

Networking

EpicClient / PartnerStatusPriority
[AUR:NETWORKING] Resolves issues around Cilium BYOCNI and VNET Integration
Since we are using Cilium as our CNI via BYOCNI on AKS (not Azure CNI), enabling VNet integration modifies the underlying Azure networking by introducing requirements for routing, which conflicts with Cilium's networking model. Without proper configuration, this disrupts connectivity, as Azure assumes default VNet routing. To resolve this, ensure that Cilium's Azure-specific configurations (e.g., IPAM mode) are properly set up, verify that the necessary Azure route tables align with Cilium's routing, adjust NSGs if needed, and use a route server to publish routes dynamically (e.g., via BGP) between the AKS subnet and the VNet to re-establish seamless connectivity.		Aurora📝 Todo🟡 Medium

Change is expected¶ Aurora is a living platform shaped by departmental demand, security requirements, operational maturity, and lessons learned through real-world adoption. Our product roadmap can change based on shifting priorities, resource availability, and unforeseen dependencies. Any item that has not yet started may be reprioritized or replaced by a higher-value initiative. Items currently in progress could be delayed as we navigate challenges and learn along the way. We share this roadmap to foster transparency, encourage collaboration, and align with our stakeholders. We welcome your feedback and contributions as we refine our path forward. 2026-01-01
